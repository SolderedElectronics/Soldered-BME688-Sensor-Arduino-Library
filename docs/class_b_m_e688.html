Detailed Description

A driver class for interfacing with the BME688 sensor.

This class provides functions to read temperature, pressure, humidity, and gas resistance from the Bosch BME688 environmental sensor. It supports different oversampling settings and operation modes.

Author
Saurav Sajeev
